# Music Maker

## Overview
Music Makerは、ユーザーが色のグリッドとインタラクトしながら音楽を作成できるウェブアプリケーションです。

## Features
- 各セルが音符を表すインタラクティブなグリッド。
- セルをクリックして色を変え、異なる音を生成します。
- 再生および停止機能を使用して、生成された音楽を聴くことができます。
